# NHS-Offers

The app displays a list of offers made available exclusively for NHS staff during the COVID-19 pandemic. The data comes from https://www.england.nhs.uk/coronavirus/nhs-staff-offers/

## Requirements

- Xcode 11.4
- iOS 13
- Swift 5

## Technologies

- SwiftUI
- Combine
- Firebase/Firestore
- AdMob
